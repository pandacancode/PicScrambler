ServerCommunication Class Reference

Inherits from NSObject
Declared in ServerCommunication.h

Overview

Basic communication class which provides facility to send requests to server. User can configure URL, request methods(GET/POST etc.), request parameters using this class.

+ getSingleInstance

Creates a singleton object of ServerCommunication class. Singleton is used to reuse single NSURLSession all the time. Return object can be used for making server calls.
